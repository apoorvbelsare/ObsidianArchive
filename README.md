# Email to Obsidian

A Python automation tool that reads emails from a mailbox and converts them into Markdown notes for an Obsidian vault.

## Features

- Reads unread emails via IMAP
- Converts emails into Markdown files
- Saves them directly into an Obsidian vault
- Runs automatically on a schedule

## Tech Stack

- Python
- IMAP
- Markdown
- Obsidian
